# This is sample project for Unity push test Nifty cloud mobile backend

# 環境

* Unity 5.2.2 + Xcode 7.3
* Unity SDK 2.1.1

# 利用

* NCMBSettingsの設定：Application Key, Client Key, Android ID
* Package name / Bundle ID: [com.nifty.push.quickstart]
